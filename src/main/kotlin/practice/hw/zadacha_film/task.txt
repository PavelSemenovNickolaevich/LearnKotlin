Задача “Фильм”
Что будете использовать (помимо пунктов из предыдущих задач):

оператор if для ветвления кода
несколько объектов на основе одного интерфейса
Текст задачи:
Создать и показать фильмы (если это возможно, согласно условиям задачи).

Сущность Film:

свойство “name”: название – String
свойство “genre”: жанр – String
функция “show”: показать фильм – выводит в консоль текст
Под сущностью подразумевается важный смысловой объект, который участвует в задаче (интерфейс + классы-реализации, либо просто отдельный класс без интерфейса)

Условия
Если у фильма отсутствует имя или название, то при попытке показать фильм – вывести ошибку и не показывать фильм:

если нет названия (null или “”): “Не указано название фильма, просмотр невозможен!”
если нет жанра (null или “”): “Не указан жанр фильма, просмотр невозможен!”
для Film должен быть создан интерфейс, т.к. это основной объект бизнес-процесса
Для сравнения String нужно использовать ==

Метод show должен вывести текст в консоль с помощью println

Какие конструкторы создавать в классе (пустые, с параметрами) – на ваше усмотрение.

Обязательно должен быть создан интерфейс, чтобы код был более расширяемым и соответствовал принципам ООП (абстракция-реализация)

Запуск
В методе main создаем несколько фильмов
1) название “Аватар”, жанр “Фантастика”
2) название “Бэтмен”, жанр “Боевик”
3) название “Терминатор”, жанр не указываем (при просмотре должно вывести ошибку, согласно условию задачи)

После этого вызываем методы show() для каждого из фильмов.


Пример вывода в консоль после выполнения программы:

Смотрим фильм: Аватар, жанр "Фантастика"
Смотрим фильм: Бэтмен, жанр "Боевик"
Не указан жанр фильма, просмотр невозможен!




Доп. задание:

Используя массив/коллекцию, создать 100 фильмов: название – индекс, жанр – “Комедия”
Проиграть все фильмы через цикл (вызвать метод show, который выводит в консоль текст)
Объекты создаем не вручную!
